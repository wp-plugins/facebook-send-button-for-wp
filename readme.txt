Add the Facebook Send Button plugin to your wordpress.

== Description ==

Add Facebook Send Button to your wordpress blog.
Options to insert send button to any posts, pages or both.
Very simple configuration.
 
== Installation ==

1. Unzip the file
2. Upload it to `/wp-content/plugins/` directory
3. Activate it from the plugins section
3. Go to the Settings->FB Send Button for customization options
